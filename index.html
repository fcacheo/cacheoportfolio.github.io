Portfolio Reflection

In my Portfolio I used a lot of CSS. CSS is extremely important when building a website because it creates all the color and style to the page. Without CSS, a website would be be boring and bland and would only show the the skeleton HTML. I personally like working with the CSS part of a website than the HTML because it allows me to be more creative. Here I can work more with colors and designs to make a webpage look nice and appealing.



I also used some bootstrap in some of my portfolio webpages. I found bootstrap to be extremely usefull when creating a webpage. It allows to used CSS aspects within your HTML code with makes things a little more manageable. Once I discovered bootstrap, I wanted to used it everywhere in my code because of how simple it is to use. It combines CSS and HTML based design so you can write less code. During my time in COSW 20, I learned a lot of web development techiques. I learned about HTML, CSS, Bootstap,JavaScript and how to use them to make a webpage. In the future, I hope that can learn more about web design/development and possibly find a career in the field.
